# LLM Wiki（SolEdu / SolaireEPDA）

本目录是**面向代理与协作者**的增量知识库：沉淀架构边界、模块入口、验证方式与可复用结论，减少每次任务从零探索仓库的成本。

## 与「人类开发者文档」的边界

| 类型 | 位置 | 说明 |
|------|------|------|
| 产品帮助、面向教师的说明 | `src/solaire_doc/` | 随应用发布，勿与本文混写 |
| 本 Wiki | `wiki/` | 开发/代理用，可随任务更新；**不**替代源码 |

权威细节以代码为准；Wiki 记录「从哪进、约定是什么、上次验证怎么跑」。

## 维护约定

1. **先读** [`index.md`](index.md) 再深入具体页面。
2. **任务结束后**：若结论可复用（接口、目录、命令、踩坑），更新对应 `modules/` 或 `architecture/` 页面；在 [`log.md`](log.md) **仅追加**一条；若新增页面则更新 `index.md`。
3. **语言**：正文使用简体中文。
4. **月度轻量校对**（可选）：对照 `index.md` 中的模块入口与 `src/solaire_doc/developer/dev-workspace.md`、主要路由文件是否仍一致。

## 根目录 `llm_wiki.md`

同目录的 `llm_wiki.md` 描述通用「LLM Wiki」理念；**本项目如何落地**以本 `README.md` 为准。
