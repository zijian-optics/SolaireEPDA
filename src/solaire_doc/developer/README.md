# 开发者文档

本目录用于沉淀随应用发布的开发者文档，适合二次开发、接口联调、部署与排障。

## 推荐阅读顺序

1. `开发约定与工程结构`：了解代码分层、接口约定与运行方式。
2. `自定义分析脚本运行机制`：了解脚本能力、输出协议与执行边界。
3. `Windows 桌面分发规划`：了解后续桌面交付、授权与环境引导策略。
4. `HTTP 接口总览`：快速查看可调用的后端接口（含学情诊断、智能助手相关路由摘要）。
5. `图谱接口（AI/脚本）`：进行自动化编排或助手编排调用。

## 文档边界

- 本目录偏工程实现与交付实践。
- 终端教师操作流程请看 `tutorial/` 和 `user/`。
