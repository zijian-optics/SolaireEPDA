# 公式编辑器与常用公式

在题库中编辑题目时，您可以用**图形化公式编辑器**输入数学式，也可以**直接手写**美元符号包裹的公式代码。下面先说明界面操作，再列出常用写法；**「效果」列在手册中会按真实排版显示**，便于对照。

---

## 一、图形化公式编辑器怎么用

**入口位置（题库编辑弹窗内）：**

- **题干**：在「内容（LaTeX）」一行右侧，点 **「插入公式」**。
- **选项**：每个选项（A、B、C…）的编辑区上方，同样有 **「插入公式」**。
- **解析、题组材料** 等带多行文本的字段旁，若有 **「插入公式」**，用法相同。

**操作步骤：**

1. 先把光标放在**要插入公式**的位置（也可先选中一段文字，插入后会替换选中内容）。
2. 点击 **「插入公式」**，会弹出标题为 **「插入公式」** 的窗口。
3. 在窗口**大输入框**里输入或编辑公式：可用**键盘**输入，也可使用界面上的**符号与结构**（与常见数学软件类似，可点选分式、根号、上下标等）。
4. 编辑完成后点击 **「插入」**：窗口会关闭，并在光标处插入一段**行内公式**（即两侧带美元符号的公式，形如 `$...$`）。
5. 若不想插入，点 **「取消」**，或按键盘 **Esc** 关闭窗口。

**说明：**

- 通过该工具插入的，统一为**行内公式**（适合夹在文字中间）。若需**单独占一行**的大公式，可在正文中手动使用**两对美元符号**（见下一节）。
- 插入后若需修改，可在原文里**直接改美元符号之间的内容**，或删掉后重新点「插入公式」。

---

## 二、手写公式：行内与单独成行

| 类型 | 写法 | 适用场景 |
|------|------|----------|
| **行内公式** | 用**一对**美元符号包裹，如 `$x^2+1$` | 与一句话在同一行 |
| **单独成行** | 用**两对**美元符号包裹多行，或占满一行的公式块 | 大公式、推导式、多行对齐 |

预览与导出试卷时，行内与单独成行会按版式规则排版；**纸面 PDF 与屏幕预览在细节上可能略有差别**，以最终 PDF 为准即可。

## 三、常用公式示例（含效果）

下表「源码」列可复制到公式编辑器或 `$...$` 中；**「效果」列为手册排版**。

| 名称 | 源码（不含外层 `$` 时，请自行加 `$` 或插入公式） | 效果 |
|------|------|------|
| 分数 | `\frac{a}{b}` | $\frac{a}{b}$ |
| 根号 | `\sqrt{x}`、`\sqrt[n]{x}` | $\sqrt{x}$，$\sqrt[3]{x}$ |
| 上下标 | `x^2`、`x_i`、`x_{10}` | $x^2$，$x_i$，$x_{10}$ |
| 希腊字母 | `\alpha,\beta,\pi,\theta` | $\alpha,\beta,\pi,\theta$ |
| 三角函数 | `\sin x,\cos x,\tan x` | $\sin x,\cos x,\tan x$ |
| 对数 | `\ln x,\log x,\log_2 x` | $\ln x,\log x,\log_2 x$ |
| 极限 | `\lim_{x\to 0}` | $\lim_{x\to 0}$ |
| 求和与积分 | `\sum_{i=1}^{n}`、`\int_a^b` | $\sum_{i=1}^{n}$，$\int_a^b$ |
| 向量 | `\vec{a}`、`\overrightarrow{AB}` | $\vec{a}$，$\overrightarrow{AB}$ |
| 括号 | `\left( \frac{1}{2} \right)` | $\left( \frac{1}{2} \right)$ |

## 四、单独成行示例（块公式）

下面为**单独成行**的展示（手册中以下块会居中显示）：

$$
\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}
$$

$$
(x+y)^2 = x^2 + 2xy + y^2
$$

（多行对齐、编号等高级环境是否能在试卷中通过，取决于导出模板；若报错可拆成多行短公式。）

## 五、常见问题

| 现象 | 建议 |
|------|------|
| 弹窗里公式对，插入后显示乱码 | 检查是否多删了美元符号；行内公式需**成对**出现。 |
| 预览正常，PDF 报错 | 个别命令在试卷引擎中可能不支持，可简化公式或拆步书写。 |
| 分式在行内太挤 | 可改为单独成行的大公式块，或缩小题干预留空间。 |

## 六、想查更多符号与命令？（扩展阅读）

**和本软件的关系**：题目在**浏览器里预览**、以及**本手册**中的公式排版，使用的是 **KaTeX**（把常见数学公式命令画在屏幕上）。它对应的是 **LaTeX 数学公式里常用的一大类写法**，**并不等于**整本 LaTeX 排版系统的全部命令。导出 **PDF 试卷**时，由另一套排版流程处理，个别命令可能在预览里正常、在 PDF 里需调整，这属于正常现象。

**若要对照「预览里到底支持哪些命令」**（与题库编辑最接近），可看 **KaTeX 官方列出的支持项**（页面为英文，站点一般可直接访问，无需特殊网络）：

- [KaTeX Supported Functions](https://katex.org/docs/supported.html)

**若希望读中文资料、系统了解数学公式写法**（面向完整 LaTeX 或日常写作，国内通常可直接打开）：

- [维基教科书 · LaTeX／数学公式（中文）](https://zh.wikibooks.org/wiki/LaTeX/%E6%95%B0%E5%AD%A6%E5%85%AC%E5%BC%8F) — 条目式说明，适合查阅符号与基本结构。
- [LaTeX 工作室](https://www.latexstudio.net/) — 中文社区与教程汇总，可搜索「公式」「符号」等关键词。

以上外链由第三方维护，若个别链接失效，可在搜索引擎中搜索 **「KaTeX supported」**、**「lshort 中文」** 等关键词，选用就近镜像或新版页面。
