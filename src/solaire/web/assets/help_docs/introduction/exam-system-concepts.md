# 组卷系统如何运作（概念说明）

本文将为您梳理组卷的核心概念，帮助您理解从“找题”到“出卷”的全流程。如果您需要具体的操作步骤，请参考左侧的其他操作指南。

---

## 考试项目里有什么

你可以把一个“考试项目”想象成一次具体的期中期末大考，或者是您个人的一个“学期教案文件夹”。在这个文件柜里，整齐地摆放着：

- **题库**：所有科目的题目、图表资源，就像超市的货架。
- **模板**：规定了试卷的结构和版式（哪类题在哪儿、分值多少、单栏还是双栏），就像购物篮。
- **导出结果**：定稿后生成的试卷文件（如 PDF 或用于分发的文件），即您打包好的战利品。

您在系统中的所有主要编辑操作，都是在这三块区域内进行流转的。

---

## 题库、模板与组卷：谁依赖谁

组卷前必须具备两个前提：**题库里有题**，以及**已有一份版式模板**。

没有题目，组卷没有内容；没有模板，就没有“这一场考”的章节与版式骨架。

1. **题库选题**：选中需要的题目，放入对应的知识小节。
2. **应用模板**：根据模板，这些题会自动排布到它们该在的地方。
3. **完成组卷**：您最终得到的是一份“选题方案”，明确记录了哪几道题进哪一节。

---

## 从选题到拿到试卷：主流程

整个组卷就像是在流水线上生产试卷，主要包括以下几步：

1. **选定模板**：先确定这次考试的骨架（例如选用“期中标准卷”的模板）。
2. **在各小节从题库选题**：往各个部分（选择题、填空题等）填入内容。
3. **设置分值与检查**：按小节统一或逐题设置分数，检查题目与资源是否齐全。
4. **生成试卷文件**：通过检查后，系统会将内容与版式结合，自动导出试卷结果（如 PDF 文件），您就可以去“导出结果”区取用了。

---

## 智能助手的辅助作用

图谱、分析与智能助手**不改变**上述主线顺序；助手可在各业务页叠加使用。如果您需要快速整理题目、或者在海量题库中定位考点，可以直接打开智能助手侧栏，通过对话来加速工作效率。
