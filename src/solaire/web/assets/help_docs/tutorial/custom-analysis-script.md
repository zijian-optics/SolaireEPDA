# 考试报表与分析工具

考试结束后，针对学生答题情况生成数据报表和掌握度分析，是我们系统中非常重要的一环。本系统内置了自动化的考后分析工具，不再需要您亲自编写复杂的数据处理代码。

这篇指南将告诉您如何在界面上选取分析工具，以及如何看懂生成的分析报表。

## 1. 如何发起考后分析

1. **进入分析界面**：在工作区左侧菜单点击 **「考试分析」**（或对应名称）进入报表工作区。
2. **选择分析目标**：选中您希望分析的考试批次。系统会自动为您抓取该次考试的总人数、题目得分率、甚至是题目挂载的“知识点”数据。
3. **选择分析工具**：在界面的工具区，您可以看到诸如“题目错误率直方图”、“知识点掌握度扇形图”、“学生成绩分层”等预设分析功能。
4. **生成报表**：点击运行后，系统将快速为您计算结果，并呈现在中部的报表区。

## 2. 认识不同种类的分析图表

### 题目错误率直方图

- **看什么**：直观展示哪些题大家错得最多。
- **怎么用**：报表上最高的几根柱子，代表了本次考试的“重灾区”。您可以在试卷讲评课上重点讲解这些对应的题目。

### 知识点掌握度分析（扇形图/雷达图）

- **前提**：生成这类图表前，您必须已经在 **知识图谱** 页面为这些题目“绑定”了对应的知识点。
- **看什么**：哪一块知识领域的失分率最高。
- **怎么用**：这不仅仅是讲评题目的依据，更是您后续教学复习的重点指针。比如“三角函数”区域的失分比例极高，说明这块需要重新进行专题复习。

### 学生成绩分层（直方图）

- **看什么**：班级里学生成绩的分布情况。
- **怎么用**：用于了解本次测验的区分度，是否有太多学生集中在某一个低分段或高分段。

## 3. 下载与导出

系统在界面上显示的各种直方图和扇形图，不仅可以直接截图，还可以：

- **下载图片**：在图表旁边点击“下载为图片”（如 SVG），这可以让您无损地插入到您的 PPT 讲义或校内报告中。
- **导出明细数据**：如果您习惯使用 Excel 继续深度加工，也可以下载结果的结构化数据表格（如 CSV 或 Excel 格式），里面包含了原始的计算分值。

## 4. 特殊分析需求

如果您需要的报表不在系统默认提供的图表类型之中，您可以请有一定开发能力的老师，或者管理员，或者智能助手，在系统的“自定义分析脚本”中为您编写专属的分析工具。一旦编写完毕，它就会像默认工具一样，作为一个按钮出现在您的操作界面上。
