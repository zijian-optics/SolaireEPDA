# 组卷流程与导出说明

本教程将引导您完成日常组卷的最核心流程：从题库中挑选题目，应用版式模板，并生成最终的试卷文件。

## 第一步：进入组卷界面

1. 在左侧导航栏点击 **「组卷」**（或对应名称）进入组卷工作区。
2. 点击 **「新建试卷」**，为本次考试起一个名称（例如“高一上学期期中考试”）。

## 第二步：选用版式模板

1. 系统会提示您选择一个 **「版式模板」**。版式模板决定了整张试卷的结构（如分为“第一卷选择题”、“第二卷非选择题”），以及排版的细节（单栏、双栏、留白等）。
2. 选中您需要的模板后，试卷的骨架就搭好了。如果不确定如何选择模板，可以参考 [版式模板与全局版式样式](./template-and-layout.md)。

## 第三步：往各小节挑选题目

1. 模板应用后，您会看到试卷分为几个具体的**小节**。
2. 展开一个小节，点击 **「从题库选题」**，界面会弹出一个题库浏览器。
3. 在题库中勾选您需要的题目，点击“右箭头”将它们加入该小节。
4. **设置分值**：您可以为小节统一设置“每题X分”，也可以针对某道特殊的题单独修改分值。

## 第四步：检查与导出试卷

1. 题目选好、分值设定完毕后，点击界面上的 **「检查并导出」**（或类似按钮）。
2. 系统会快速检查是否有遗漏的内容（比如分值未设置、题目插图丢失）。
3. 检查通过后，系统会自动将您挑选的内容“倒入”模板进行排版，最后生成供打印和分发的 **PDF** 文件。
4. 导出成功后，您可以在项目目录的 **「导出结果」**（exams）区域获取它。

---

## 小贴士

- 如果在生成的试卷上发现**错别字**，请返回 **题库** 修改该题，然后再重新点击一次“导出”，错别字就会自动更正。不要试图用其他软件强行修改导出的 PDF。
- 导出的 PDF 是标准的高清打印格式。系统会自动导出一份**带答案和解析的教师版**和一份**留空白的学生版**，无需您手工删减题目。
