# 版式模板与全局版式样式

如果您已经习惯了将一道题和它的格式混在一起排版的系统（例如用 Word 调节字体大小和缩进），这部分功能会带给您一种截然不同——且更省时省力——的体验。

## 1. 什么是版式模板？

您可以将**版式模板**理解为一套预先做好的“空试卷”。
在这份“空试卷”里，已经定义好了：

- **纸张和栏数**：比如，是单栏还是双栏？
- **各部分结构**：例如“第一部分：单项选择题（15小题）”、“第二部分：非选择题”。
- **各题型的默认说明**：在每个大题前会有像“以下题目每题提供四个选项，只有一个最符合题意”这样的统一说明语。

当您在组卷时应用一个版式模板，系统会自动根据模板留好的“坑”，将题库里的题目填充进去。

## 2. 全局版式样式有什么用？

在版式模板的基础上，您可能还会对整个试卷的细节有些微调，比如字号、行距、页脚、或者是标题的内容。这就是**全局版式样式**起作用的地方。

- **全局修改**：如果您想把试卷里的选择题选项调开一点，或者把全部题目的字体放大，您只需要在“全局版式样式”里改一个设置即可。它会自动应用到试卷里的**所有题目**上。
- **不再逐题排版**：如果您在某一题的文本里遇到需要加粗或者居中的字眼，也可以在题库里用简单的标记处理。但试卷主干的美观度，我们强烈建议交给“版式模板”和“全局版式样式”统一接管。

## 3. 选择版式模板的流程

1. 在组卷界面的开始，您会看到一个**「版式模板」**的下拉列表。
2. 里面会有预置的几种常用样式，例如“期中标准卷（双栏）”、“随堂测验卷（单栏）”。
3. 选定后，您的组卷大纲将立刻被切分成几个小节，您只需要往里填题目即可。
4. 当您需要为同一批题“换一种感觉”时，重新选择另一个版式模板，系统将在导出时为您自动进行完全不同的排版。

---

如果您的教学需求比较特别，现有的版式模板不能满足要求（比如，需要定制带有特定学校 Logo 或水印的排版），您可以请相关的技术老师或者管理员帮助定制jinja模板，或者请求智能助手为您生成。系统支持高度可定制的版式生成，但在日常使用中，您只需要专注挑选适合的模板就可以了。
