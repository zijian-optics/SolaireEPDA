# 知识图谱页使用说明

侧栏进入 **知识图谱** 后，可在画布上查看与编辑项目内的知识点结构。

## 基本操作

- **浏览**：拖动画布平移；使用界面提供的缩放控件（或触控板手势，视环境而定）放大缩小。
- **节点**：圆点或卡片表示知识点；节点大小等视觉提示可能与关联资料数量等**业务含义**相关，便于区分重点。
- **连线**：节点之间的连线表示关系类型（如前置、相关等），具体类型以服务端与图例为准。
- **编辑**：选中节点后可查看详情、修改名称或描述、调整在画布上的位置；保存后会在下次打开时保持。

## 题目与资料

- 可将题库中的题目**绑定**到某个知识点，便于从「知识」视角反查题目。
- 可为知识点**挂载资料**（如项目资源目录下的文件）；上传或引用路径需符合项目约定，且文件已存在于当前项目内。

## 与自动化的关系

若需通过 HTTP 批量写入节点、关系或绑定，请参阅本手册 **「图谱 HTTP 接口（自动化）」** 篇；日常使用以界面操作为主即可。
