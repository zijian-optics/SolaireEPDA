# 知识图谱与思维导图

知识图谱的主要业务价值在于建立**题目与知识点**之间的关联。通过构建知识框架并将其与题目绑定，系统能在考后自动为您生成每一位学生、每一个班级的“知识点掌握度”分析报表。

为了方便您快速搭建学科大纲，系统现已全面升级，支持**按科目独立管理**，并提供**思维导图**与**网状知识图谱**两种无缝切换的视图。

---

## 1. 界面结构概览

进入知识图谱页后，界面分为三个主要区域：

- **左侧（科目栏）**：管理不同的学科图谱。您可以点击切换科目，或者在底部点击“新建图谱”。
- **中部（画布区）**：您在这里搭建和浏览知识框架。顶部工具栏可以用来切换“思维导图”或“知识图谱”视图。
- **右侧（详情面板）**：当您选中某个知识点时，这里会显示它的详细信息、挂载的题目和资料。

---

## 2. 两种视图的用法

您可以随时在画布顶部的工具栏中，切换当前图谱的显示方式。底层的数据是同一套，只是展现和编辑的方式不同。

### 视图一：思维导图（推荐日常搭建使用）

这是每次打开图谱时的默认视图。它以树状结构从左向右展开，非常适合您快速录入教材的章节目录或知识大纲。

**💡 快捷键高效录入：**

- **双击节点**：直接修改节点名称。
- **按 `Tab` 键**：在当前选中的知识点下，快速创建一个“子知识点”。
- **按 `Enter` 键**：在当前选中的知识点同级，快速创建一个“兄弟知识点”。
- **按 `Delete` / `Backspace` 键**：删除选中的知识点。

您可以利用这些快捷键，像打字一样飞速建立起一门课的完整知识树。

### 视图二：网状知识图谱

当知识点之间存在复杂的交叉关系（例如：“要想掌握抛物线，必须先掌握二次函数”），您可以切换到“知识图谱”视图。

- **添加复杂关系**：在顶部工具栏点击“增加关系”，然后依次点击画布上的两个节点，即可将它们连线，并在右侧面板选择它们的关系类型（如“前置”、“相关”）。

---

## 3. 核心业务：挂载题目与资料

在画布中**点击选中**任意一个知识点后，右侧会弹出详情面板，面板上方有三个页签：

1. **节点信息**：您可以在这里修改知识点的名称、关系、标签。
2. **关联题目**：您可以将题库中的题目**绑定**到这个知识点上。这是生成考后“知识点掌握度”报表的**必须步骤**。
3. **关联资料**：如果您为这个考点准备了专门的复习课件或讲解文档，可以将其挂载在这里，方便以后统一管理和查阅。
4. **笔记**：您可以在笔记栏目里随时添加、编辑和删除笔记，便于知识点回顾。

---

## 4. 批量操作与自动化辅助

如果您需要从外部的 Excel 表格或文本大规模导入全校的知识点大纲，可以使用智能助手侧栏，通过自然语言对话的辅助能力快速导入，避免逐个节点手动敲击创建。
