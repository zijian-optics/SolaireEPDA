# 智能助手侧栏

在组卷、题库、模板、知识图谱、考试分析等页面，您都可以打开**智能助手侧栏**，用自然语言描述需求，由系统在项目范围内调用已开放的能力（如整理题目说明、辅助维护图谱、运行分析等）。

## 使用要点

- **须先打开考试项目**：助手只能访问当前项目内的题库、模板、导出结果与图谱数据。
- **模型配置**：首次使用前，请在界面提供的设置中配置大语言模型服务（具体项以当前版本界面为准）；未配置时无法发起对话。
- **建议与执行**：部分操作可先以「建议」形式给出方案，再由您确认是否写入项目；涉及改动的步骤一般会经过确认，避免误改。
- **与当前页配合**：您在某一业务页打开助手时，系统会结合**当前所在页面**生成简短背景说明，使回答更贴场景；侧栏收起后，同一会话仍可继续，重要进度会在界面角落提示。
