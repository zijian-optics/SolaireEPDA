# 题库交换包：备份、分享与搬运题目

您可以把某一题集里的题目**打成一个交换包**，方便备份、发给同事，或搬到另一份项目里用。也可以把别人给的交换包**导入**到您指定的科目和题集。

---

## 导出（从当前项目拿出题目）

1. 在软件中进入**题库**，选中要导出的**科目**和**题集**。
2. 使用界面上的 **「导出当前题集为交换包」**（或类似名称）按钮。
3. 下载得到的文件保存好即可。

**关于图片：**
题目中的插图会自动打包在一起，换台电脑导入后图片依然能正常显示。

---

## 导入（把题目放进当前项目）

1. 在**题库**中使用 **「从交换包导入」**（或类似名称）。
2. 选择您得到的交换包文件。
3. 按提示选择要写入的 **科目** 和 **题集**（若目标题集已存在，请留意是否覆盖或合并，以软件提示为准）。

**说明：**
为保证导入效果，建议您优先使用本系统导出的交换包。导入其他来源的文件时，系统会自动进行格式检查并给出指引。

---

## 音频等资源

交换包中可能预留**音频**等扩展支持。当前若题目涉及音频，请以软件当时版本的说明为准；导入后若有文件缺失，界面一般会给出提示。

---

## 常见问题

| 问题 | 说明 |
|------|------|
| 导入后图片不显示 | 看导入结果里的提示；可能是包内缺图。 |
| 能否只单独拷某道题 | 可以，但如果有插图，需确保插图也被正确带过来；直接用交换包通常更省事。 |
