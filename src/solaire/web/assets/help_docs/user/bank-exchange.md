# 题库交换包：备份、分享与搬运题目

您可以把某一题集里的题目**打成一个压缩包（ZIP）**，方便备份、发给同事，或搬到另一份项目里用。也可以把别人给的压缩包**导入**到您指定的科目和题集。

---

## 导出（从当前项目拿出题目）

1. 在软件中进入**题库**，选中要导出的**科目**和**题集**。
2. 使用界面上的 **「导出当前题集为交换包」**（或类似名称）按钮。
3. 下载得到的 **ZIP 文件**保存好即可。

**包里大致包含：**

- 合并好的题目正文（一道文件里包含本题集全部题目）。
- 题目里用到的**图片**（在压缩包内的图片文件夹中）。
- 一份**说明信息**（记录导出时间、来源题集等，一般无需手改）。

题目正文里的插图在包内用**相对路径**引用，别人用同一软件导入后可以正常显示。

---

## 导入（把题目放进当前项目）

1. 在**题库**中使用 **「从交换包 ZIP 导入」**（或类似名称）。
2. 选择您得到的 ZIP 文件。
3. 按提示选择要写入的 **科目** 和 **题集**（若目标题集已存在，请留意是否覆盖或合并，以软件提示为准）。

**说明：**

- 使用本软件导出的标准包，导入最稳定。
- 若压缩包结构不标准（例如只有题目文件、没有完整说明），软件会尽量兼容，但可能弹出**提示**；请按提示检查是否有图片未找到等情况。

---

## 音频等资源

交换包中可能预留**音频**等目录，用于以后扩展。当前若题目涉及音频，请以软件当时版本的说明为准；导入后若有文件缺失，界面一般会给出提示。

---

## 常见问题

| 问题 | 说明 |
|------|------|
| 导入后图片不显示 | 看导入结果里的提示；可能是压缩包内缺图，或路径与旧版不一致。 |
| 能否只拷一个题目文件 | 可以，但插图需一并放到对应资源目录；用交换包通常更省事。 |
